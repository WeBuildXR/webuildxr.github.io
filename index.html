WeBuildXR

Turn any browser into a world building stage.

Open source

This library can be used for any purpose at no cost. Let us do the heavy lifting for you!

Collaborative platform

WeBuildXR allows you to generate custom geometry, and many design options including color, transparency and more.

Browser based

The code of this library is 100% Javascript and runs on the client side, not depending on servers.

Lightweight

The builds can be included in any project as compact common js or ES modules, taking less than 20000 KB.

Maintainable

The platform is self-contained, and allows you to add world building and viewing to any project.

Custom API

You can use this as an efficient world builder or as a world viewer in your web app.
